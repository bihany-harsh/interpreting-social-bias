## Description

The data generated here is a subset of the data as presented in the $\text{IG}^2$ paper [[link]](https://arxiv.org/abs/2406.10130). As is the case, the code is borrowed, modified and inspired from the code the authors have published.

Since we aim to work with decoder-only transformers, v1 of all the runs concerned work with prompts which end in predicting the demography towards the end of the sentence.

- 11/17 templates are employed in v1.
- To generate the data, run the `generate_data_v1.py` file.

### Generalizations

The subsequent generalizations would include having StereoSet and Shades working in here.
